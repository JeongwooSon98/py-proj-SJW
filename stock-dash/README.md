# 📈 My Stock Dashboard

관심종목 15개를 매일 **여러 보조지표가 지금 같은 방향을 가리키는지** 한 화면에서 확인하는 대시보드입니다.
일봉 기준이며, 지표 계산과 채점은 **완성된 봉**으로만 합니다. (투자 참고용)

---

## 🔴 시작하기 전에 — 이 앱이 해주는 것과 안 해주는 것

이 대시보드는 **종목을 골라주지 않습니다.**

15년 일봉(하락장 포함) 백테스트에서 **종합 점수 +3 이상의 10일 초과수익은 +0.1%p** 였습니다. 아무 날에나 산 것과 통계적으로 구분되지 않는 수준입니다. 개별 신호도 대부분 ±0.5%p 이내(노이즈 범위)입니다.

| | |
|---|---|
| ✅ **이 앱이 하는 일** | "지금 이 종목에서 MACD·스토캐스틱·RSI가 동시에 매수 쪽을 가리킨다"는 **사실 요약**. 15개 종목 × 10개 지표를 매일 눈으로 훑는 수고를 없애 줍니다. |
| ❌ **이 앱이 못 하는 일** | "이 종목이 오른다" / "지금 사면 된다". 점수는 **예측값이 아니라 합의도**입니다. |

그래서 UI 어디에도 "적극 매수", "오늘 매수 적합" 같은 **행동 권유 문구가 없습니다.** 등급 이름은 "매수신호 강함/있음"처럼 **신호가 얼마나 모였나**만 말합니다. 판단은 사용자 몫입니다 — 대신 판단에 필요한 재료(신호 발생 시점, 현재 진입 위치, 검증 결과, 내 과거 기록)를 전부 눈에 보이게 깔아 줍니다.

## 실행

```bash
npm install
npm run dev      # 개발 서버
npm run build    # 빌드
npm run backtest # 터미널에서 백테스트 (앱 패널과 동일 엔진)
```

브라우저만 있으면 됩니다. 계정·API 키·서버가 필요 없고, 데이터는 Yahoo Finance에서 직접 받아옵니다.

## 화면 구성 — 위에서 아래로 5개 영역

### ① 지수 패널 — 오늘 시장의 온도

KOSPI·KOSDAQ·NASDAQ·S&P 500·다우 시세와 USD/KRW 환율, 그리고 **시장 레짐 요약** 한 줄이 뜹니다.

- **VIX**: < 20 위험 선호 / 20~30 중립 / > 30 공포
- **ADX(14)**: ≥ 25 추세장 / < 20 횡보장 (방향이 아니라 추세의 *강도*)

> 💡 **응용**: ADX가 낮은 횡보장이면 아래 카드에서 RSI·볼린저 같은 **평균회귀 신호**가, 높은 추세장이면 MACD·정배열 같은 **추세추종 신호**가 더 의미 있게 읽힙니다. 이 위젯은 점수에 반영되지 않는 참고 정보입니다.

### ② 📊 신호 종합 패널 — **이 앱의 핵심**

관심종목 15개를 10개 신호로 채점해, **매수 쪽(+3 이상) / 매도 쪽(−3 이하)** 으로 신호가 모인 종목만 카드로 보여줍니다. 중립(−2~+2)은 아예 안 뜹니다.

카드 하나에 담긴 정보:

| 요소 | 무엇을 말하나 |
|---|---|
| **점수 칩** `+8점 ▸` | 신호 합의도. 누르면 **항목별 가·감산 내역 팝업** (반대 방향 신호까지 전부) |
| **오늘의 상태 배너** | 점수 강도 + 진입 위치를 합친 **한 줄 서술** (예: `매수신호 강함 · 20일선 근처`) |
| **신호 배지** | 충족한 신호와 점수. **hover = 왜 이게 신호인지 설명**, **클릭 = 차트의 그 날짜로 이동·강조** |
| **신선도 표기** | `당일 발생` / `3봉 전 · 이후 +6%` — 며칠 전 신호를 지금 보고 있는 건 아닌지 |
| **진입 위치 배지** | 🟢 20일선 근처 / 🟡 다소 확장 / 🔴 과열·추격 / ⚠️ 낙폭과대 |
| **리스크 계획** | ATR 기반 손절(2×ATR) · 목표(3×ATR) · 손익비 · 변동성 |
| **🆕 / ▲2 델타** | 직전 완성 봉엔 없던 신호 / 어제 대비 점수 변화 |
| **📝 기록** | 이 진입을 매매일지에 박제 |

> 💡 **가장 실용적인 사용법**: 매일 이 패널의 **`🔁 직전 완성 봉 대비` 요약**만 보세요. "새로 ±3에 진입한 종목 / 목록에서 빠진 종목"을 알려주므로, 15개를 다시 훑을 필요 없이 **바뀐 것만** 확인하면 됩니다.

> 💡 **점수만 보고 사지 마세요.** 점수는 "몇 개 지표가 겹쳤나"일 뿐입니다. 같은 +8점이라도 `🟢 20일선 근처`와 `🔴 과열·추격 · 5봉 전 신호 · 이후 +12%`는 완전히 다른 자리입니다. 카드가 이 둘을 **일부러 분리해서** 보여주는 이유입니다(점수로 합치면 검증 안 된 가중치가 숨어들기 때문).

**점수 체계 요약** (자세히는 → [INDICATORS.md](INDICATORS.md))

가중치는 **타이밍(±3) > 위치(±2) > 추세·확인 보조(±1)** 위계를 따릅니다.

| 분류 | 신호 | 점수 |
|---|---|:---:|
| 타이밍 | MACD 골든/데드크로스 (0선 필터로 신뢰도 차등) | ±3 / ±2 |
| 타이밍 | 스토캐스틱 과매도/과매수 강크로스 | ±3 |
| 위치 | RSI 과매도/과매수, 볼린저 하단/상단(%B), 다이버전스 | ±2 |
| 보조 | 정배열, 거래량 급증, 주봉 추세, 상대강도(RS), 레짐 정합 | ±1 |

만점 **매수 +17 / 매도 −16**. 등급컷 **강함 ±7 / 있음 ±3**.

### ③ 🔬 신호 검증 (간이 백테스트) — 점수를 의심하는 도구

**같은 화면 안에 반증 도구가 들어 있습니다.** 관심종목 15개 × 15년 일봉으로 위 신호들이 과거에 실제로 맞았는지 검증합니다.

맨 위에 **📏 기준선(전체 봉 진입)** 이 고정으로 뜹니다. 신호와 무관하게 아무 날에나 샀을 때의 성과입니다.

> 💡 **반드시 셋째 줄 `초과 ±X%p`를 보세요.** 첫째 줄 절대수익률은 **알파가 아니라 베타**입니다 — 이 종목들(NVDA·AAPL…)은 아무 날에나 사도 10일 뒤 평균 +1.1%가 나옵니다. 초과 ≤ 0이면 회색 `⚠` 로 표시되며, "아무 날에나 산 것만도 못하다"는 뜻입니다.

5가지를 검증합니다: **개별 신호별 성과** / **점수 구간별 성과** / **ATR 청산 성과**(카드가 표시하는 손절·목표 규칙 자체의 기대 R) / **지연 진입 성과**(며칠 늦게 사면?) / **레짐 분해**(강세장 vs 약세장 — 진짜 우위라면 둘 다 (+)여야 합니다).

셀에 **마우스를 올리면** 중앙값·표준편차·최악값이 뜹니다. 평균만 보면 "가끔 −15% 터지는 신호"를 놓칩니다.

⚠️ 해석 주의: 관심종목이 **살아남은 대형주**라 생존 편향이 있고, 보유기간이 겹쳐 표시된 n은 독립 표본 수가 아닙니다. 여기 수치는 **가장 좋게 봐준 상한선**입니다.

### ④ 📝 매매일지 — 측정되지 않은 유일한 것

백테스트가 말하는 건 "이 10개 신호를 **기계적으로 합산하면** 우위가 없다"이지, **"당신의 판단에 우위가 없다"가 아닙니다.** 화면을 보고 실적·뉴스·업황을 얹어 내리는 재량 판단은 한 번도 측정된 적이 없습니다. 그걸 재는 유일한 방법이 기록입니다.

- **진입**: 카드의 📝 기록 → 그 순간의 **점수·신호 라벨·진입 위치·1R(손절폭)을 그대로 박제**합니다(나중에 재계산 안 함). 메모에 재량 근거를 남기세요.
- **청산**: 청산가·사유를 넣으면 수익률·**R 배수**·보유일이 자동 계산되고, **같은 기간 벤치마크(SPY/KOSPI) 대비 초과수익**이 함께 박제됩니다.
- **집계**: 평균/중앙 수익률 · 승률 · 기대 R · **평균 초과** · 표본수(n).
- **백업**: JSON 내보내기/가져오기로 브라우저가 날아가도 복구됩니다.

> 💡 **응용 — 이 앱의 진짜 쓰임새**: 30건쯤 쌓이면 "점수 +7 이상에서만 들어간 거래"와 "메모에 실적 근거를 적은 거래"의 **평균 초과수익을 비교**해 보세요. 내 재량이 신호보다 나은지, 아니면 그냥 시장을 따라간 건지가 그때 처음으로 보입니다.
>
> ⚠️ `n < 30`이면 앱이 **"표본 부족 — 결론 내지 말 것"** 경고를 띄웁니다. 절대 수익률만 보면 베타를 알파로 착각합니다. 사후에 합격 기준을 만들지 마세요("이번 건만 빼면 좋았는데").

### ⑤ 관심 종목 — 차트로 확인

15개 종목의 현재가·대비·등락률. 행을 펼치면 **4개 차트**가 뜹니다.

1. **가격**: 캔들 + MA5/20/60/120 + 볼린저 밴드(20, ±2σ)
2. **RSI(14)**: 70/30 기준선 + 다이버전스 마커(◇)
3. **Slow Stochastic(14,3,3)**: 과매도/과매수 강크로스 마커(▲▼)
4. **MACD(12,26,9)**: MACD·Signal·히스토그램 + 다이버전스 마커

기간(1달~3년)·봉 종류(일봉/주봉)를 바꿀 수 있고, 미국 개별주 6종(NVDA·TSLA·AAPL·MSFT·GOOGL·AMZN)은 차트 위에 **펀더멘털 참고 박스**(PER·PBR·PEG·배당·애널리스트 목표가와 상승여력)가 뜹니다. 점수에는 반영되지 않는 맥락 정보입니다.

> 💡 신호 카드의 배지를 클릭하면 **여기로 자동 스크롤되어 그 날짜에 세로선이 그어집니다.** 점수를 믿지 말고 차트에서 직접 확인하라는 뜻입니다.

## 응용 시나리오

**아침 5분 루틴**
1. 지수 패널에서 오늘 레짐 확인(VIX·ADX) →
2. 신호 종합 패널의 `🔁 직전 완성 봉 대비`로 **어제와 달라진 종목만** 확인 →
3. 관심 가는 카드의 **배지를 클릭**해 차트에서 근거 눈으로 확인 →
4. 진입한다면 **📝 기록**으로 그 순간의 점수·근거를 박제.

**"이 신호 믿어도 되나?" 싶을 때**
백테스트 패널을 펴서 그 신호 행의 **초과수익**과 **레짐 분해**를 보세요. 대부분 ±0.5%p 이내로 나옵니다 — 그게 정직한 답입니다.

**"지금 들어가도 되나?" 싶을 때**
점수가 아니라 **진입 위치 배지(⑬)와 신선도 표기(⑫)** 를 보세요. `🔴 과열 · 5봉 전 신호 · 이후 +12%`면 신호는 맞았지만 **그 자리는 이미 지나갔습니다.**

**분기마다**
매매일지 집계에서 **평균 초과수익**을 확인하세요. 절대 수익률이 아무리 좋아도 초과가 0 근처면 시장을 따라간 것뿐입니다.

## 알아둘 동작

- **장중에 점수가 안 움직이는 건 정상입니다.** 진행 중인 오늘 봉은 잘라내고 **완성된 봉**으로만 채점합니다(카드에 `🕐 장중 · 신호는 {날짜} 종가 기준` 배지). 미완성 봉으로 채점하면 오후에 본 골든크로스가 종가에 사라져(repaint) "근거를 확인한다"는 전제 자체가 깨집니다. 차트·현재가만 실시간입니다.
- **데이터는 5분 캐싱**, `↻ 새로고침`으로 강제 갱신. 백테스트는 무거워서 펼칠 때 1회 계산 후 30분 캐시.
- **관심종목 변경**은 [src/StockDashboard.jsx](src/StockDashboard.jsx)의 `WATCHLIST` 배열을 편집합니다(한국 종목은 `005930.KS` 형식).
- **데이터 출처**: 시세·차트·점수 = Yahoo Finance(비공식, corsproxy.io 경유) / 펀더멘털 = FMP 빌드타임 정적 스냅샷(`npm run fetch:fundamentals`로 갱신).
